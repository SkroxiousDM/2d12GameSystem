# Core Glossary

A consolidated, alphabetical reference for the core terms used across the 2d12 ruleset. Each entry is a short, rules-facing definition; see the chapter cross-referenced in the entry for the full rules.

| Term | Definition |
|---|---|
| Action | The unit of activity spent on a turn. Each character has a number of Actions per turn equal to their Alacrity. A minor action costs 1 Action; a major action costs 2 Actions. See Ch 11. |
| Action Roll | A roll used to resolve uncertain actions, including attacks, skill checks, and some spell uses. It uses 2d12, compares the result to a Challenge Threshold, and may produce success, failure, complications, or critical outcomes. See Ch 1. |
| Advantage | A favorable modifier to a roll. In action rolls, it is usually represented by rolling one extra d12 and keeping the better pair, though talents and special rules can also grant it. |
| Alacrity | A derived stat: ((AGI + INS) / 2) + Traits. It sets the number of Actions a character gets each turn and feeds Initiative. See Ch 4. |
| Armour | A protective rating applied to a hit location. Armour reduces rolled Damage at that location before the remainder is applied to Vitality. Armour values are determined by worn gear (see Ch 15.1) and may be modified by Quality, Flaws, magic, or Traits. |
| Attack of Opportunity | See Opportunity Attack. |
| Attribute | A core character rating: **Might (MIG)**, **Constitution (CON)**, **Dexterity (DEX)**, **Agility (AGI)** (physical), and **Reason (REA)**, **Insight (INS)**, **Charisma (CHA)**, **Willpower (WIL)** (mental). Attributes determine many checks and feed derived values. See Ch 4. |
| Background | A character's life experience and station prior to play, chosen during character creation. A Background grants attribute points, skill ranks, talents, equipment, and connections. See Ch 2. |
| Challenge Threshold (CT) | The target number for an action roll. The GM sets it based on task difficulty and circumstance. |
| Characteristic | A stat used in opposed or other checks. Functionally interchangeable with Attribute or derived stat in most rules text; not a separately formalized category. |
| Common Test | A check used when nobody directly opposes the acting character but the outcome still matters. |
| Complication | A result where the action succeeds or fails with a cost, twist, or other negative consequence. Complications can also feed scene resources such as Doom. |
| Concealment | A Stealth skill specialization. Covers hiding oneself, others, or objects to avoid detection. See Ch 5. |
| Concentration | A focus state required to maintain spells, rituals, and certain ongoing effects. Concentration is automatically lost when the character becomes Staggered, takes a Critical Wound, or fails a relevant Resilience (Discipline) test under duress. See Ch 3, Ch 11. |
| Condition | A named status effect that changes what a character can do or how they function. Conditions may be Minor or Major. See the Conditions Appendix (Ch 91). |
| Courage | A derived stat: (WIL − 2) + Morale + Traits. Used to anchor leaders and unit Morale checks. See Ch 4. |
| Critical Failure | A disastrous action roll result in which both dice match on a failure. It causes severe consequences, including Doom gain and, for attacks, a Fumble roll. |
| Critical Hit | A strong attack result that causes a Critical Wound even if the target has not exceeded maximum Vitality. |
| Critical Success | A powerful action roll result in which both dice match on a success. It grants a strong benefit, including Fortune gain, and can improve attack results. |
| Critical Wound | A severe injury suffered when excess damage is taken, or when an attack or effect inflicts one directly. Critical Wounds have location-based effects and can lead to death if untreated. See Ch 11 and the Critical Wounds table. |
| Culture | The wider society a character was raised in, chosen during character creation. Culture establishes baseline traits, languages, and starting skill access. See Ch 2 and 2b. |
| Damage | The amount of harm inflicted by an attack or effect before mitigation by armour or other defenses. Resolved as Nd3 − N, plus the relevant attribute bonus, minus armour. Physical damage reduces Vitality; mental damage reduces Resolve. See Ch 11 and Ch 15. |
| Deadly | A weapon trait that increases the severity of any Critical Wound the weapon inflicts. See Ch 15.2. |
| Defence | A derived stat: (CON − 2) + Armour + Cover + Traits. Sets the passive difficulty for some attacks against the character. See Ch 4. |
| Desperate | A condition applied to NPCs when the party's Superiority exceeds their Resolve. It can cause them to flee, surrender, or make a final attack, depending on context. See Ch 11. |
| Desperate Action | A bonus action granted by certain Traits or Talents. A character may use any number of distinct Desperate Actions on a turn in addition to their regular Actions; each Desperate Action may be used once per Rest. See Ch 11. |
| Disguise | A Stealth skill specialization. Covers using clothing, props, and body language to convincingly assume another's appearance. See Ch 5. |
| Doom | A GM resource gained through certain failures, complications, and critical failures. |
| Doom token | A unit of Doom held by the GM. |
| Drain | A damage effect or consequence in which any unsoaked Resolve stress causes Fatigue. |
| Environment | The surrounding location and circumstances of a scene — terrain, weather, lighting, crowd, structure. The environment can grant Cover, impose Difficult Terrain, hide threats, or carry Environmental Traits. See Ch 19. |
| Environmental trait | A persistent feature of a scene's environment (e.g. choke point, slick floor, blinding glare, sacred ground) that modifies tests, movement, or actions while in effect. Identifying environmental traits is one way to gain Superiority in combat. See Ch 11, Ch 19. |
| Evasion | A passive defensive value used to set the difficulty for some attacks. Base Evasion is 10 + Dexterity bonus + Traits, and certain Talents allow it to be calculated from Fighting Skill Rank instead while wielding specific weapon configurations. See Ch 4, Ch 6. |
| Extended Check | A check resolved over multiple rolls, used for complex or time-consuming tasks with a threshold and possible deadline. |
| Fate | A resource the party may gain from successful rolls or other reward cycles. |
| Fate Point | A reward or meta-resource that may be granted by the GM in place of some success rewards. |
| Fatigue | A cumulative condition representing exhaustion, deprivation, or despair. Each point temporarily reduces Stamina, Vitality, and Resolve by 1. |
| Fear Check | A test made when a character is exposed to a frightening creature, scene, or supernatural presence. Usually a Resilience (Discipline) test against a CT set by the source of the fear. Failure inflicts the Frightened condition or, in severe cases, Madness or Resolve damage. See Ch 14, Ch 16. |
| Fighting | The skill used for combat actions, melee attacks, and some force-based checks. |
| Fortune | A player resource gained through certain critical successes or other rewards. |
| Fortune point | A unit of Fortune held by a player. |
| Full Rest | A 24-hour rest in a safe location. It restores all spent Stamina, all spent Resolve, all spent Power, and heals some Wounds and ability damage. See Ch 8. |
| Fumble | A consequence of a Critical Failure on an attack roll: roll on the Fumble table for the type of weapon used. Outcomes range from dropped weapons and broken straps to friendly-fire and self-injury. See the Fumble table. |
| GM | The gamemaster, who sets difficulties, adjudicates uncertain outcomes, and manages NPC and scene resources. |
| Hit Location | The body part struck by an attack, usually determined by the Complication die. Current locations include head, arms, legs, and body. |
| Immobile | A condition state where a character cannot move from their current position, though they may still take other actions. Distinct from Restrained, which is more severe. See Ch 14, Ch 91. |
| Initiative | The order in which characters act in combat, determined by 1d12 plus Alacrity. |
| Leadership | A Manipulation skill specialization. Covers commanding others, issuing clear orders, and maintaining morale. See Ch 5. |
| Level of Success (SL) | A measure of how well a check succeeded or failed beyond the threshold. It can be spent on benefits such as information, speed, or confidence. |
| Long Rest | An 8-hour rest that restores some Stamina, Resolve, and Vitality, and improves recovery through first aid. See Ch 8. |
| Madness | A condition gained when a character takes Sanity damage or exceeds their Sanity threshold. Manifests as a specific named affliction rolled or chosen from the Madness tables. See Ch 16. (See also: Irrational, the immediate condition imposed when Resolve reaches 0.) |
| Morale | A trait value that anchors Courage and feeds unit-level morale. A leader's Morale typically substitutes for that of their followers when the leader is present. Loss of Morale precipitates the Desperate condition. See Ch 4, Ch 11. |
| NPC | A non-player character. Any character run by the GM, including allies, enemies, bystanders, and creatures. |
| Opportunity Attack | A melee attack made with Disadvantage when an adjacent enemy willingly moves away from you. Costs no Action and discourages careless withdrawal. See Ch 11. |
| Opposed Test | A check where two characters roll against each other, and the higher result or higher Level of Success wins. |
| Power | A magical rating used for spells and rituals, determining how many damage or ritual dice a character rolls. Its formula varies by spellcasting tradition: Traditional (2 + INS + Traits), Research (2 + REA + Traits), Dabbling (1 + WIL + Traits). See Ch 3, Ch 4. |
| Power source | The cosmological wellspring from which a spellcaster draws magical energy (e.g. cult patron, oracular tradition, druidic compact, runic lore). The Power source determines available spell lists, ritual access, and certain narrative constraints. See Ch 3. |
| Pursuit | A structured scene resolving a chase. Each round, every participant chooses a maneuver in secret, the GM reveals or generates an obstacle, and the maneuvers are resolved. The quarry may flee on foot, by mount, or in a wagon; the pursuer may chase, cut off escape, block the way, or shoot. See Ch 10. |
| Push | Rerolling a failed check at a cost of Stress, Vitality, or Resolve depending on the attribute used. See Ch 1. |
| Reaction | An action taken outside a character's turn, usually in response to a trigger. Reactions cost action economy and sometimes Stamina depending on the rule. |
| Resolve | A derived mental durability stat (3 + WIL + Traits) used to resist fear, mental damage, and pressure. When it reaches 0, the character loses 1 Sanity and becomes Irrational. See Ch 4, Ch 16. |
| Resolve resistance | A target's effective resistance to social pressure or mental damage, equal to their current Resolve. Compared against accumulated Social Superiority to determine when an NPC becomes Desperate in social conflict. See Ch 12. |
| Rest | A period of recovery, including Short Rest, Long Rest, and Full Rest. See Ch 8. |
| Role | A character's adventuring archetype, chosen during character creation. The Role grants starting Talents, skill access, and shapes a character's initial niche. See Ch 2. (Distinct from NPC Roles such as Mook/Brute/Caster/Boss; see Ch 20.) |
| Sanity | A derived mental resilience limit (REA + INS + CHA + Traits). Sanity damage and exceeding the Sanity threshold trigger Madness. See Ch 4, Ch 16. |
| Skill | A trained capability tied to an attribute and used for checks. Skills can have ranks and specializations. See Ch 5. |
| Social Conflict | A structured scene where one character tries to get something from another through words, leverage, fear, negotiation, or persuasion. See Ch 12. |
| Social Superiority | A social momentum resource gained and spent during Social Conflict. It can improve social checks and may grant Advantage when spent in sufficient quantity. |
| Specialization | A focused sub-area of a skill. When it applies, it grants Advantage to the related check. |
| Speed | A movement value used to determine how far a character can move on their turn. Base speed is 30 ft. unless otherwise noted. |
| Spellcaster | A keyword or character trait indicating access to magic-related talents, skills, and rules. |
| Staggered | A condition gained when Vitality reaches 0. It reduces Alacrity, breaks Concentration, and makes the character more vulnerable to further injury. See Ch 91. |
| Stamina | A physical exertion resource (MIG + CON + DEX + AGI + Traits) used for movement, carrying, and other strenuous actions. See Ch 4. |
| Stamina Invested | Stamina committed to carrying gear rather than spent temporarily. It represents encumbrance and returns when the load is reduced. |
| Stress | A cost paid when pushing a failed check and a broader resource or consequence tied to mental and physical strain. |
| Superiority | A shared momentum resource representing advantage in combat or social conflict. It is gained through preparation and success, and spent for bonuses or pressure. See Ch 11, Ch 12. |
| Talent | A character feature that provides passive or active benefits, often with prerequisites. Talents can grant actions, reactions, desperate actions, or modifications to attributes and skills. See Ch 6. |
| Threshold | A target amount that must be reached in an extended check. |
| Trait | A character feature that can affect attributes, derived stats, or requirements. Traits come from Culture, Background, Role, Talents, ancestry, and circumstance, and are added into many derived-stat formulas. |
| Truth | In ritual magic, a fact established about the working that increases the difficulty of subsequent skill tests for that ritual by 1. Disruptors and adversaries may add Truths to a ritual instead of inflicting a complication directly. See Ch 3.9. |
| Vitality | A physical health resource (3 + CON + Traits). Damage reduces Vitality first; when it reaches 0, the character gains a Critical Wound and becomes Staggered. See Ch 4. |
| War Magic | The category of spells designed for the battlefield — direct combat applications cast in rounds rather than as rituals. War Magic is the practical opposite of Ritualistic Magic (Ch 3.9), and is the default mode of the Runic, Oracular, and Druidic spellbooks (Ch 3.1, 3.2, 3.4). |
| Wound | A major injury tracked separately from Vitality. A character has a maximum number of Wounds (3 + CON + Traits), and exceeding that limit can be fatal. See Ch 4. |
