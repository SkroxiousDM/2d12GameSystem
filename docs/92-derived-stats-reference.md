# Derived Stats Reference

A one-page reference for every derived statistic in the game: how it is calculated, where it is spent, and how it is recovered. Use the source chapter for full rules; use this sheet to fill out a character sheet or to settle a question mid-scene.

## Quick Formula Card

| Derived Stat | Formula | Source |
|---|---|---|
| **Stamina** | MIG + CON + DEX + AGI + Traits | Ch 4 — Attributes |
| **Alacrity** | ((AGI + INS) / 2) + Traits | Ch 4 — Attributes |
| **Defence** | (CON − 2) + Armour + Cover + Traits | Ch 4 — Attributes |
| **Vitality** | 3 + CON + Traits | Ch 4 — Attributes |
| **Wounds (max)** | 3 + CON + Traits | Ch 4 — Attributes |
| **Courage** | (WIL − 2) + Morale + Traits | Ch 4 — Attributes |
| **Resolve** | 3 + WIL + Traits | Ch 4 — Attributes |
| **Sanity** | REA + INS + CHA + Traits | Ch 4, Ch 16 — Sanity |
| **Power (Traditional)** | 2 + INS + Traits | Ch 3, Ch 4 |
| **Power (Research)** | 2 + REA + Traits | Ch 3, Ch 4 |
| **Power (Dabbling)** | 1 + WIL + Traits | Ch 3, Ch 4 |
| **Carrying Capacity** | Standard items ≤ Stamina | Ch 4 — Attributes |
| **Initiative** | 1d12 + Alacrity | Ch 4, Ch 11 |
| **Movement (per action)** | 1 zone; 2nd costs 2 Sta; 3rd costs 3 Sta | Ch 4, Ch 11 |
| **Damage on a hit** | Roll (N + SL)d3 − (N + SL), + Attribute Bonus, − Armour; N is the weapon's Damage rating | Ch 11, Ch 15 |

| **Madness gain trigger** | Resolve = 0 | Ch 4, Ch 16 |

## Worked Example: A Mid-Career Character Sheet Line

> **Adelheid the Quartermaster.** MIG 3, CON 4, DEX 3, AGI 3, REA 2, INS 4, CHA 3, WIL 3. Traits: 3.
>
> - **Stamina:** 3 + 4 + 3 + 3 + 3 = **16**
> - **Alacrity:** ((3 + 4)/2) + 3 = 3 + 3 = **6** *(round down)*
> - **Defence:** (4 − 2) + 2 (mail) + 0 + 3 = **7**
> - **Vitality:** 3 + 4 + 3 = **10**
> - **Wounds (max):** 3 + 4 + 3 = **10**
> - **Courage:** (3 − 2) + 0 + 3 = **4**
> - **Resolve:** 3 + 3 + 3 = **9**
> - **Sanity:** 2 + 4 + 3 + 3 = **12**
> - **Power (Traditional):** 2 + 4 + 3 = **9**
> - **Initiative roll:** 1d12 + 6
>
> She carries 12 Standard items invested (≤ 16 Stamina). Her remaining 4 Stamina is the pool she draws from for extra movement, Pushed rolls, and Catching Breath recovery during a fight.

## Cap Reminders

- No attribute exceeds **7** by any means.
- An attribute may not exceed the character's **number of Traits**.
- A skill may not exceed its governing attribute's natural cap.
- Defence and Courage have no hard cap, but the GM should question any value above 12 outside of magical and narrative spikes.
- Wounds remaining at 0 with Vitality at 0 and Resolve at 0 is a candidate for Fate burn (Ch 4, Ch 7).
